# CodeAlpha_FlashcardQuizApp

An interactive, responsive 3D Flashcard Quiz web application built using pure HTML, CSS, and JavaScript. This platform provides complete client-side data persistence (CRUD) allowing users to dynamically customize their flashcard decks for real-time studying. Developed during my **Web & App Development Internship** at **CodeAlpha**.

---

## 🚀 Features
* **3D Flip Transitions:** Built using unique CSS transform properties to recreate realistic card-flipping mechanics.
* **Linear Array Navigation:** Seamless "Next" and "Previous" pagination logic with edge-case handling.
* **Full CRUD Customization:** Users can seamlessly write new custom question-answer card objects or delete them via form input states.
* **LocalStorage State Caching:** Saves any added or modified flashcards directly to the browser's storage cache so data isn't lost on page reloads.

---

## 🛠️ Built With
* **HTML5:** Framework layout structures for interactive quiz sections and control inputs.
* **CSS3:** Configured using modern custom flexbox layouts, perspective 3D variables, animations, and responsive dimensions.
* **JavaScript (ES6+):** Handling DOM click events, LocalStorage JSON serialization, indexing arrays, and interface render workflows.
